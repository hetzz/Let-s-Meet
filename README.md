### Features

- Establish a peer to peer video connection with all basic functionalities
- Ability to create and join rooms for video calls.
- Building a signalling server.

### Installation

``` sh
- git clone https://github.com/hetzz/P2P.git
```
- `npm install`.
- run `npm run start`
- run `npm run android`
